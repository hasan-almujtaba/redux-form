# Redux Form

Form handling with RTK Implementation

## Live Demo

[Live Demo](https://redux-form-h.netlify.app)

## Usage

### Development

Start local environment

```bash
pnpm dev
```

### Build

To build the App, run

```bash
pnpm build
```

### Code Linting

To lint the App, run

```bash
pnpm lint
```

### Code Formatting

To format the App, run

```bash
pnpm format
```
